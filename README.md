# Решение задач из курса осеннего практикума для студентов 3-го курса кафедры Алгоритмических языков ВМК МГУ
**Язык программирования: Common Lisp**

Привет! В этом репозитории собраны решения домашних заданий практикума, а также основная информация о том, как устроен курс

## Структура курса:
*Получение условий и сдача кода проходят лично с преподавателем.*

2 домашних задания: 
- Небольшое задание 1, направленное на реализацию 5 несложных алгоритмов, с дополнительными задачами во время сдачи
- Большое задание 2, направленное на реализацию многофункционального консольного приложения, с реализацией дополнительных возможностей во время сдачи
  - К реализации задания 2 необходимо подготовить развернутый отчет по выполненной работе 

*Важно учитывать: варианты заданий для каждого студента индивидуальные, на выбор варианта предоставляется время*

## Система оценивания:
- После каждого занятия студентам предлагается написать самостоятельную работу, что также позволяет отслеживать посещаемость
  - Оценки за самостоятельные суммируются в единую оценку
  - При получении высокого балла за самостоятельные работы, оценка за контрольную работу выставляется автоматом
- Общая оценка по курсу складывается из 4 частей, каждая из которых оценивается по 5-балльной шкале:
  - Самостоятельные работы
  - Контрольная работа
  - Задание 1
  - Задание 2
- Поздняя сдача заданий дедлайна постепенно снижает получаемую оценку за курс вплоть до **Удовлетворительно**. Старайтесь сдавать задачи как можно раньше!
  
## Структура репозитория:

Задания из курса разбиты по папкам согласно следующему шаблону:
```text
    ├── task1
    │ ├── task1.lsp
    │ └── lisp1_25.pdf
    │
    ├── task2
    │ ├── task2_geom.lsp
    │ └── lisp2_25.pdf
    │
    ├── report
    │ ├── report_lisp_ovl.pdf
    │ └── report_lisp_ovl.tex
```
где taskN*.lsp - реализация задания N, lispN_25.pdf - условие задания N, report_lisp_ovl.pdf - готовый отчет, report_lisp_ovl.tex - исходный LaTeX-код отчета
